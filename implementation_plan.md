# Convert Project to Full AI/ML Video Analysis System

The project currently uses a simulated AI dashboard with CSS-based bounding boxes and hardcoded mock data. This plan will transition it to a fully functional AI/ML application that processes real video streams and generates actual insights using Computer Vision.

## Proposed Architecture

1. **AI Model**: We will integrate **YOLOv8** (via the `ultralytics` package), which is a state-of-the-art real-time object detection model.
2. **Video Streaming**: We will replace the static HTML5 `<video>` elements with MJPEG (Motion JPEG) streams. The backend will read frames using `OpenCV`, run them through YOLOv8, draw real bounding boxes, and stream the processed frames to the frontend.
3. **Dynamic Alerts**: When the AI detects specific objects of interest (e.g., people in restricted areas, weapons, etc.), it will generate real alerts and store them in memory. The frontend will fetch these via the `/api/alerts` endpoint.
